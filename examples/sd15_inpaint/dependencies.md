# Dependencies

## Models

### Checkpoints
- `models/checkpoints/dreamshaper_8.safetensors` → [Download (Civitai)](https://civitai.com/models/4384/dreamshaper)

## Custom Nodes

No custom nodes required — all nodes are ComfyUI built-in.

## Input Images

Prepare two images (same resolution):
- `photo.png` — the image you want to inpaint
- `mask.png` — black/white mask where white = area to inpaint
